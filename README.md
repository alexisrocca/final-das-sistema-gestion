# FINAL DAS - Sistema de Gestion

### [!] Diagrama de clases presente en [DAS - DC.pdf](DAS%20-%20DC.pdf)

Este proyecto utiliza **SQL Server Express _LocalDB_**. 
Si desea usar **SQL Server Express** (no LocalDB), cambie el string de conexion en [App.config](FINAL_POO_Sistema_de_Gestion/App.config) 
* Server=(localdb)\MSSQLLocalDB  ->  Server=.\SQLEXPRESS

### Si desea utilizar LocalDB
Si al ejecutar la aplicación recibes un error de conexión, asegúrate de tener instalada y activa la instancia local:
- **Instalación**: Verifica en *Visual Studio Installer* > *Visual Studio* > *Modificar* que tienes instalado el componente **SQL Server Express LocalDB**.
