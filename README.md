# FINAL DAS - Sistema de Gestion

Este proyecto utiliza **SQL Server Express LocalDB**. Si al ejecutar la aplicación recibes un error de conexión, asegúrate de tener instalada y activa la instancia local:
- **Instalación**: Verifica en *Visual Studio Installer* > *Visual Studio* > *Modificar* que tienes instalado el componente **SQL Server Express LocalDB**.
